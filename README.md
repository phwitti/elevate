# elevate
Command-line application to open an elevated cmd in the current directory.

## Usage

(elevate.exe should be in some place that's included in your PATH variable for any sensible use)

1. Use the cmd in your daily life
2. Be in any directory (i.e. c:/some/random/directory/)
3. Need an elevated cmd (i.e. for mklink usage)
4. Type 'elevate'
5. Have an additional elevated cmd open in your current directory
